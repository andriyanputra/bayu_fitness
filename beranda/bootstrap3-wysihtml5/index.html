bootstrap-wysihtml5
Simple, beautiful wysiwyg editors - for bootstrap 3

About

bootstrap-wysihtml5 is a javascript plugin that makes it easy to create simple, beautiful wysiwyg editors with the help of wysihtml5 and Bootstrap 3

View project on Github

Usage

$('.textarea').wysihtml5({
    "font-styles": true, //Font styling
    "emphasis": true, //Italics, bold, etc
    "lists": true, //(Un)ordered lists
    "html": true, //Allow edit HTML
    "link": true, //Allow to insert link
    "image": true, //Allow to insert image
    "color": true //Change font color
    });
